# EGGIE
Team: Somang Lee, Huynh Nguyen, Katelyn Davis

A vibecoding project featuring a hard-boiled egg timer and a scrambled egg cooking mini-game. 

## Overview

- **Hard-Boiled Egg Timer**: Choose from three cooking modes (Soft, Medium, Hard) with visual countdown, progress tracking, and audio alerts
- **Scramble Game**: A multi-step cooking game to keep you occupied while you wait for your timer to countdown.


## Tech Stack

### Core Framework & Language
- **React JS**
- **TypeScript** 
- **Vite** 
- **Tailwind CSS**

### Styling & Design
- **Custom Assets handrawn in Procreate** 
- **Web Audio API** 
- **Custom hooks**
